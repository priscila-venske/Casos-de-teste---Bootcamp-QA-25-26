# Projeto QA Manual — E-commerce

Projeto realizado durante o Bootcamp TripleTen.

## Objetivo
Validar funcionalidades críticas de um sistema de e-commerce.

## Tipos de teste executados
- Testes funcionais
- Testes de regressão
- Testes de interface
- Validação de API no Postman

## Ferramentas utilizadas
- Jira
- Postman
- Google Sheets
- Android Studio

## Atividades realizadas
- Criação de casos de teste
- Execução de testes manuais
- Reporte de bugs
- Criação de checklist
- Documentação de resultados
